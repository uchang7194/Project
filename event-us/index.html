맞춤형 행사지원 플랫폼 EVENT-US

'이벤터스'는 성공적인 행사를 위한 스마트 파트너입니다.

EVENT-US 개요

누구나 클릭 몇 번이면 행사에 필요한 기능들과 디자인을 갖춘 전문 플랫폼을 만들 수 있는 행사전용 클라우드 서비스입니다.

  • cloud service

    클라우드 서비스로써 행사전용 플랫폼을 제공하여 지속적으로 사용가능

  • powerful solutions

    행사에 필요한 대부분 프로세스들을 대체해줄 ICT 솔루션들을 제공

  • modern design

    Google Material Design을 기반으로 깔끔하고 간편한 UX/UI 제공

  • easy customize

    누구나 쉽게 행사에 최적화되게 기능 설정이나 디자인 변경이 가능

강력한 특장점

30여가지의 다양하고 강력한 솔루션들 제공
행사 커스터마이징 서비스

  • 참가자 모집/관리 솔루션

    • 1. 행사 참가신청
    • 2. 출석확인/현장등록
    • 3. 문자/이메일 발송
    • ...

  • 실시간 소통 솔루션

    • 1. 질의응답
    • 2. 투표 + 결과
    • 3. 설문조사/퀴즈
    • 4. 채팅
    • ...

  • 다양한 이벤트 솔루션

    • 1. 경품 추천
    • 2. 모의투자/전문심사
    • 3. 스탬프/코인 미션
    • 4. 1대100 퀴즈
    • 5. 실시간 경매
    • ...

  • 통계 보고서 솔루션

    • 1. 각 기능별 통계 대쉬보드 제공
    • 2. 각 기능별 원본 데이터 파일 제공
    • ...

이벤터스 행사 참가자 모집/관리 솔루션 설정방법 영상 보기

이벤터스 행사 현장용 솔루션 설정방법 영상 보기
